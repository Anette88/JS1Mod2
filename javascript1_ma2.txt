The constructor method is a method for creating and initialising an object created within a class.
There can only be one method with the name constructor in a class.

5. Briefly describe scope in JavaScript.

Scope refers to where we define variables and functions. 
Global scope happens when you define a variable prior to writing any JavaScript.
Can be accessed from everywhere in your code.	
Local scope is defined inside a function. Can be accessed only from inside that function.
